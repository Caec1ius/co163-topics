Meltdown and Spectre

An insight into two critical security vulnerabilities affecting modern processors

Meltdown and Spectre are vulnerabilities which affect modern processors. Both vulnerabilities leverage the microarchitectural design of processors, allowing processes to access memory locations which would usually not be permitted[1]. As a result, attackers are able to read sensitive memory locations and target a wide range of affected devices, including computers, cloud servers and smartphones.
